# i18n Architecture

## Core Flow
- Setup globally managed via `next-intl`. 
- JSON files stored in `/messages/en.json` and `/messages/ar.json`. These dictionaries must remain 100% matched in structural nesting hierarchy.

## Usage Strictness
- `useTranslations('Namespace')` fetches string interpolations directly inside React client components.
- Manually defining copy variables inside a `COPY` object to sidestep JSON dictionaries is an architectural violation and will fail the spec enforcement. 
- Zod schema values mapping to translation functions (e.g., `tValidation('emailRequired')`) must inject `tValidation` directly into their wrapping hooks (`useMemo`) to trigger reactive rebuilds when the user toggles languages on the fly. 
- URL-level mapping guarantees automatic prefixation (`[locale]/domain/...`).
